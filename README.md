# ChangelogX

## Installation

Clone the repository and change into the directory:

```bash
git clone
cd ChangelogX
```

Install the dependencies:

```bash
npm install
```
## Commands

Running Locally:

```bash
npm run dev
```

Running in Production:

```bash
npm run build
npm run start
```

Testing:

```bash
npm run test
```

Linting:

```bash
npm run lint
```
