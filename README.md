# ChangelogX

## Installation

Clone the repository and change into the directory:

```bash
git clone
cd ChangelogX
```

Install the dependencies:

```bash
npm install
```

## Commands

Running Locally:

```bash
npm run dev
```

Running in Production:

```bash
npm run build
npm run start
```

Testing:

```bash
npm run test
```

Linting:

```bash
npm run lint
```

## Project Structure

```bash
src\
    | - config\ # Configuration files
    | - handlers\ # Request handlers
    | - modules\ # Authentication and Middleware
    | - routes.ts\ # Routes
    | - server.ts\ # Server
    | - db.ts\ # Database connection
    | - index.ts\ # Entry point
```
